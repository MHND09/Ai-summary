AI-Summary API Documentation

This documentation provides details about the available endpoints in the AI-Summary API, which uses Google's Generative AI to process and analyze images and text content.

Summarize Whiteboard Image

POST /api/summarize

Description

Analyzes an image of a whiteboard and provides a concise summary of its content.

Request

Expects a JSON payload with the following fields:

Field Type Description Required
imageUrl String URL of the whiteboard image to analyze Yes

Example Request

{
    "imageUrl": "https://example.com/whiteboard-image.jpg"
}

Response

Returns a JSON object with the following fields:

Field Type Description
summary String A brief description of what's on the whiteboard (limited to 50 words)

Example Response

{
    "summary": "The whiteboard shows a: flowchart of a user authentication process with login, registration, and password recovery paths, including database interactions and security validation steps."
}

Error Responses

Status Description
400 Missing imageUrl in request
500 Unable to generate summary due to server error

Extract Table from Whiteboard Image

POST /api/extract-table

Description

Extracts and formats content from a whiteboard image, preserving the structure and formatting.

Request

Expects a JSON payload with the following fields:

Field Type Description Required
imageUrl String URL of the whiteboard image to extract content from Yes

Example Request

{
    "imageUrl": "https://example.com/whiteboard-table-image.jpg"
}

Response

Returns a JSON object with the following fields:

Field Type Description
tableContent String The extracted content with original formatting preserved

Example Response

{
    "tableContent": "Project Timeline\n\nPhase 1: Research | Jan - Feb\nPhase 2: Development | Mar - May\nPhase 3: Testing | Jun - Jul\nPhase 4: Deployment | Aug"
}

Error Responses

Status Description
400 Missing imageUrl in request
500 Unable to extract table content due to server error

Classify Content

POST /api/classify

Description

Classifies text content by assigning the most appropriate labels from a provided list.

Request

Expects a JSON payload with the following fields:

Field Type Description Required
content String The text content to classify Yes
availableLabels Array of Strings List of possible labels to choose from Yes

Example Request

{
    "content": "A study on the effects of climate change on polar bear populations in the Arctic",
    "availableLabels": ["Science", "Environment", "Politics", "Technology", "Health", "Education"]
}

Response

Returns a JSON object with the following fields:

Field Type Description
labels Array of Strings List of relevant labels assigned to the content

Example Response

{
    "labels": ["Science", "Environment"]
}

Error Responses

Status Description
400 Missing content or availableLabels in request
500 Unable to classify content due to server error
